# Zenodo-upload
A program that can be used to upload you file to Zenodo

Upload Command:

```python
python zenodo-upload.py --id Your_id --dir /path/to/the/upload-dir/
```

Example:

```python
python zenodo-upload.py --id 10996591 --dir ./ADD_2022
```
